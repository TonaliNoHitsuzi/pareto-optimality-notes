# 帕累托最优：从理论到工程实践

一份系统性的帕累托最优（Pareto Optimality）学习笔记，涵盖经济学起源、数学形式化、经典与现代算法、工程应用，以及完整的 NSGA-II 实战案例与交互式可视化。

## 项目简介

本项目以"学术入门书籍"的口吻撰写，保持理论严谨性的同时，通过直观类比、数学推导与工程实例，帮助具备基础数学或计算机背景的读者顺畅建立从概念到实践的完整知识链条。

全书共七章，从 Vilfredo Pareto 的福利经济学思想出发，经由多目标优化的数学骨架，推进到 NSGA-II / MOEA / MORL 等算法工具箱，最终落地于一个真实的三目标物流网络优化案例——用纯 JavaScript 从零实现 NSGA-II，并用 RBF 平滑曲面可视化帕累托前沿。

## 配套代码

第七章配套了一个完整的、零第三方依赖的 JavaScript 实现，位于 `code/` 目录：

| 文件 | 说明 |
|------|------|
| `config.js` | 五城经纬度 → Haversine 距离矩阵、需求量与参数表 |
| `nsga2.js` | NSGA-II 核心：评估、修复、快速非支配排序、拥挤距离、分段遗传算子、主循环 |
| `delaunay.js` | 自写 2D Delaunay 三角化（Bowyer-Watson 增量算法），用于凸包判断 |
| `main.js` | 运行入口：执行优化，输出 `pareto_result.json` |
| `build_viz.js` | 读取结果，生成综合可视化页面（RBF 平滑曲面 + 投影 + 收敛曲线） |

## 快速开始

环境要求：Node.js（本项目使用 v24，无需安装任何依赖）。

```bash
cd code
node main.js        # 运行 NSGA-II（约 0.4 秒），输出 pareto_result.json
node build_viz.js   # 生成综合可视化页面 pareto_visualization.html
```

随后在浏览器中打开 `pareto_visualization.html` 即可。

## 可视化

综合可视化页面（基于 Plotly.js）包含四个视图：

- **三维帕累托前沿曲面**：用高斯径向基函数（RBF）将散点插值为光滑曲面，Viridis 色阶按未覆盖率着色，底部投影等高线，可拖拽旋转
- **三张二维投影**：成本-碳排、成本-未覆盖、碳排-未覆盖的两两权衡曲线
- **收敛曲线**：三个目标最优值随迭代代数的变化（归一化）
- **拐点高亮**：端点连线距离法识别的性价比转折点

## 实战案例

**长三角绿色物流网络设计**：某电商企业在上海、杭州、苏州、南京、宁波五城中选址建设区域配送中心（RDC）与前置仓，并决定运输车辆类型，同时优化三个目标：

| 目标 | 含义 | 求解结果范围 |
|------|------|-------------|
| $f_1$ | 总成本（建设 + 运输） | 10,176 – 14,500 万元 |
| $f_2$ | 碳排放量 | 0 – 1,032 吨 CO₂ |
| $f_3$ | 24 小时未覆盖率 | 0% – 53% |

NSGA-II（种群 100，迭代 500 代）在约 360ms 内求得 59 个帕累托最优解，清晰呈现了"低成本-高碳排-低覆盖"到"高成本-零碳排-全覆盖"的完整权衡前沿。

## 技术栈

- **算法**：纯 Node.js / ES，NSGA-II 全部从零实现，无第三方优化库
- **可视化**：Plotly.js（CDN 引入），RBF 插值曲面
- **数学工具**：自写高斯消元、2D Delaunay 三角化、径向基插值
